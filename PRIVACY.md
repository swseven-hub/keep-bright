# 隐私说明

Keep Bright 是一个原生 macOS 菜单栏工具，不收集、上传或出售任何个人数据。

## 本地数据

应用会使用 `UserDefaults` 在本机保存这些偏好：

- 保持时长
- 自定义保持时长
- 防睡眠模式
- 启动后是否自动开启保持亮屏
- 低电量电池保护模式
- 电池保护阈值
- 插电后是否自动恢复保持亮屏
- 是否每天自动检查更新
- 是否已经显示过首次启动引导

这些数据只保存在你的 Mac 上。

## 网络访问

如果启用了“每天自动检查更新”，应用会访问 GitHub Releases API，检查是否有新版本。你也可以在菜单栏里手动触发检查更新。

应用不会向自建服务器发送数据，也不包含分析、广告或追踪 SDK。

## 通知

应用会在开启、关闭、定时结束、低电量保护触发等场景发送本地系统通知。通知内容由应用在本机生成。

如果 macOS 的通知预览设置被关闭，系统可能只显示“收到一条通知”这类泛化文案。Keep Bright 无法绕过系统隐私设置；你可以在“系统设置 > 通知”中调整通知预览。
