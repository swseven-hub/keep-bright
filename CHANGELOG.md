# 更新日志

## v1.7.0

### 更新内容

- 新增“自动化”偏好设置页，可独立开启指定 App、全屏、外接显示器和连接电源自动保持亮屏。
- 新增指定 App 规则，支持按前台 App 名称或 Bundle ID 匹配，默认提供 Keynote、PowerPoint、Zoom、腾讯会议、QuickTime Player 等常见演示和会议应用。
- 新增全屏窗口检测，进入全屏演示、观影或会议时可自动开启保持亮屏。
- 新增外接显示器检测，连接外接屏或投影仪时可自动开启保持亮屏。
- 新增电源适配器检测，插电时可自动开启，拔电后按规则恢复。
- 菜单栏面板新增自动化状态显示，可以直接看到当前触发规则或暂停状态。
- 自动化开启后会在触发条件结束时自动关闭；如果用户手动关闭，会暂停当前自动化触发直到条件结束，避免反复抢回控制。
